## 📈 GA4管理画面での確認方法

### 1. リアルタイム確認
1. **GA4管理画面** → **リアルタイム**
2. サイトにアクセスしてユーザー数が表示されるか確認
3. ページビューがリアルタイムで記録されるか確認

### 2. イベント確認
1. **GA4管理画面** → **イベント**
2. Instagram DMボタンをクリック
3. `instagram_dm_click` イベントが記録されるか確認
4. カスタムパラメータ（`button_location: 'header'` 等）が表示されるか確認

ブラウザの開発者ツールのコンソールで以下が表示されます：
